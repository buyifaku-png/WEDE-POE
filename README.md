# Mzansi Tech Repairs

A multi-page website built for **Mzansi Tech Repairs**, a fictional tech repair business based in Gqeberha, Eastern Cape, South Africa. This project was developed as the Part of Evidence (POE) submission for **WEDE5020 — Web Development**, The Independent Institute of Education (IIE).

## Project Description

Mzansi Tech Repairs offers fast, reliable, and affordable repairs for smartphones, laptops, and other electronic devices. This website allows customers to learn about the business, browse services, book a repair, and get in touch with either of two branches in Gqeberha.

## Pages

- **Home (`index.html`)** — Hero section, live search, "Why Choose Us," FAQ accordion, and an image gallery with lightbox.
- **About Us (`aboutus.html`)** — Company history, mission, and vision.
- **Products (`products.html`)** — Repair services overview, filterable product grid, and an image gallery with lightbox.
- **Book a Repair (`enquiry.html`)** — Booking form with real-time JavaScript validation.
- **Contact Us (`contactus.html`)** — Branch details, contact form with validation, and an interactive map with a branch selector.
- **References (`references.html`)** — Technical references used during development, in IIE referencing format.

## Features

- Responsive design (mobile, tablet, and desktop layouts)
- Royal Blue / Black / White colour scheme with Roboto and Calibri typography
- Live search/filter feature
- Image gallery with a custom-built JavaScript lightbox (next/prev/close, keyboard support)
- FAQ accordion using vanilla JavaScript
- Interactive Google Maps embed with a branch switcher and "Get Directions" link
- Client-side form validation on both the Enquiry and Contact forms, with inline error messages
- On-page SEO: descriptive title tags, meta descriptions, meta keywords, and descriptive image alt text

## Technologies Used

- HTML5
- CSS3 (Flexbox and Grid)
- Vanilla JavaScript (no frameworks or libraries)
- Google Maps Embed API

## How to Run This Project

1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Install the **Live Server** extension
4. Right-click `index.html` and select **"Open with Live Server"**.
5. Alternatively, double-click `index.html` to open it directly in your browser.
